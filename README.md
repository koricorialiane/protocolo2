# Protocolo Titán — Dashboard profesional GSM/EDGE

Aplicación Streamlit para analizar una red táctica GSM/EDGE en dos escenarios:

- Convoy ferroviario de alta velocidad: Doppler, tiempo de coherencia, fading Rayleigh/Rician, presupuesto de enlace y margen.
- Campamento base: reutilización celular, reparto ARFCN, C/I co-canal, capacidad Erlang B y certificación por RBW.

## Ejecución local

```bash
pip install -r requirements.txt
streamlit run streamlit_app.py
```

## Despliegue en Streamlit Cloud

Main file path:

```text
streamlit_app.py
```
